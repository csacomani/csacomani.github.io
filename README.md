# Portal Inovat - GitHub Pages

Pagina estatica de apresentacao do Portal Inovat, pronta para publicacao em
`https://csacomani.github.io` e no dominio configurado em `CNAME`.

## Estrutura

- `index.html` - conteudo da landing page
- `styles.css` - estilos responsivos
- `assets/` - logo e imagem da interface mobile

Nao ha etapa de build. O GitHub Pages pode publicar diretamente a branch
principal deste repositorio.
